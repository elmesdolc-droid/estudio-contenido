# Fonts

Tipografías compartidas por el motor de overlay, empaquetadas en el propio
repo (no dependen de lo que tenga instalado cada ordenador).

- `PlayfairDisplay-Variable.ttf` — serif con carácter para marcas
  artesanales/premium (Instagram, El Mes Dolç). Fuente variable: el peso
  (Regular/Medium/SemiBold/Bold/ExtraBold/Black) se ajusta en código vía
  `overlay.font_weight` en `configs/<perfil>.json`, no hace falta un
  fichero por peso.
  Fuente: [Google Fonts / Playfair Display](https://github.com/google/fonts/tree/main/ofl/playfairdisplay)
  (SIL Open Font License, ver `PlayfairDisplay-OFL.txt`), gratuita para uso
  comercial.

Wallapop sigue usando Arial (`C:\Windows\Fonts\arialbd.ttf`) por legibilidad
neutra en anuncios de segunda mano.
